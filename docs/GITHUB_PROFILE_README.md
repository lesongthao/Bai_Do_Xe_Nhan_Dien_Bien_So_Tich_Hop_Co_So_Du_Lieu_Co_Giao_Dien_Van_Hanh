## Featured Project

### Smart Parking System using Arduino Uno R3

Built a smart parking prototype that detects vehicles, recognizes license plates, validates vehicle data, controls servo barriers, and stores entry/exit history.

**Project highlights**

- Developed Arduino Uno R3 firmware to read LM393 sensors and control servo barriers.
- Implemented UART/Serial communication between PC software and Arduino.
- Integrated USB camera capture with a Python desktop application.
- Built a license plate recognition pipeline using YOLOv8n, OpenCV, PaddleOCR, and post-processing.
- Designed an SQLite database for valid plates, parked vehicles, and entry/exit history.
- Built a Tkinter UI for camera preview, recognition results, parking status, and logs.
- Achieved **97.50% plate-level accuracy** on 200 labeled test images.
